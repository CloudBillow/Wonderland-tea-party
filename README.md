# Wonderland tea party v1.0.0

这是一个 ai 聊天应用, 使用 nuxt3.js 开发的前后端半一体化应用(为什么要叫半呢, 因为数据库需要手动配置).

目前支持 `通义千问` 和 `deepseek`.
